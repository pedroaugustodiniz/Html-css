Principais formatações

Negrito ou destaque

Nesse frase temos um em negrito usando a tag B (não semantica)

Nessa frase, temos essa em destaque usando a tag STRONG (semantica)

Itálico / Ênfase

Nessa frase temos em Itálico como destaque Usando a tag I (não semântica)

Nessa frase temos em Itálico como destaque Usando a tag EM (semântica)

Texto marcado

Temos marcação usando a tag MARK.

Texto Grande e Pequeno

Temos big para maior ( ele está absoleto) e temos SMALL para letras pequenas.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a TAG DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS.

Existe também o sublinhado com a Tag U ( não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 (sem inserir ) inserindo x 20+3 usando a Tag SUP.

Texto subcristo

Para inserir coisas do tipo H2O ( sem inserir ) inserindo H 2O usando a Tag SUB.
